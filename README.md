## Vite React TypeScript Tailwind Starter

A barebones template for bootstrapping simple frontend projects.

Also comes with ESLint and Prettier for auto-formatting on save.

### Dependencies
* Vite 2.7.6
* React 17.0.2
* TailwindCSS 3.0.7
* TypeScript 4.5.4
* ESLint 8.5.0

### Getting Started

```bash
npm install

npm run dev
```
